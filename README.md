# 2. FORZIP서버 입장방법
1. 카카오톡 서버에 입장한다.
2. %인증마크 을(를)쓴다
3. 인증 문자가 갠으로 날라온다(갠 불가시 입장 불가)
4. 화이트리스트(이)가 적용되었을시 입장가능
5. ~~핵, 테러 진적시 밴~~
6. **입장가능**
